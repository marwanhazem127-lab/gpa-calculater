# 🎓 GPA Calculator System

A smart GPA Calculator developed using **C++** that helps students calculate their GPA efficiently based on subject grades and credit hours.

The system allows users to enter subjects, credit hours, and grades, then automatically calculates the final GPA using grade points and weighted averages. 🚀

# 🔄 How It Works
The program stores subjects using a structured data model containing:
- Subject Name
- Credit Hours
- Grade

Each grade is converted into grade points, then multiplied by the credit hours to calculate total points.

Finally, the GPA is calculated using:

GPA = Total Points / Total Credit Hours ⚡
